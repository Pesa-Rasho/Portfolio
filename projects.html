Projects I have done

Garment Factory Management & Optimization System

This system is designed to streamline the management and operations of a garment factory. It handles inventory tracking, production scheduling, and worker allocation, ensuring efficient workflow and resource optimization. The system also provides real-time data analysis to help factory managers make informed decisions.

Key Features

  • Inventory Management: Real-time tracking of raw materials and finished goods.
  • Production Scheduling: Automatic scheduling based on order priority and resource availability.
  • Worker Management: Efficient allocation of workers to different tasks.
  • Analytics Dashboard: Provides insights into production efficiency and resource usage.

Technologies Used

  • Front-end: React, HTML5, CSS3
  • Back-end: Node.js, Express
  • Design: Figma

Foodie

Foodie is a platform designed to connect chefs with viewers who are passionate about cooking. Chefs can post their recipes, while viewers can explore, watch, and learn how to prepare a wide variety of dishes. The platform allows easy navigation of recipes by categories and enables users to favorite and interact with their preferred content.

Key Features

  • Chef Dashboard: Chefs can post and manage their recipes.
  • Recipe Viewing: Viewers can browse, watch, and save favorite recipes.
  • Favorites: Save favorite recipes to a personalized favorites page.
  • Category Navigation: Browse recipes by cuisine, type, or difficulty.
  • Responsive Design: Seamless user experience across devices.

Technologies Used

  • Front-end: React, HTML5, CSS3
  • Backend: Firebase for authentication and database
  • Version Control: Git, GitHub
  • Design: Figma
